# RC Car Project

An Arduino-based Bluetooth Controlled RC Car that can be operated using an Android mobile application.

## Features

- Bluetooth Control
- Forward, Backward, Left and Right Movement
- Stop Function
- Easy to Build
- Low Cost

## Components

- Arduino Uno
- L298N Motor Driver
- HC-05 Bluetooth Module
- DC Motors
- RC Car Chassis
- Wheels
- Battery Pack
- Jumper Wires

## Circuit Diagram

![Circuit](hero_rc_car.png)

## Arduino Code

The Arduino sketch is available in:

```
arduino/rc_car.ino
```

## How to Use

1. Upload the Arduino code.
2. Connect the HC-05 module.
3. Pair the Bluetooth module with your phone.
4. Open a Bluetooth RC Car app.
5. Drive the car.
